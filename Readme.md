# Prac_UIvPRPO_4 (Вариант №6)

Программа для нормализации телефонных номеров в текстовых файлах.  
Приводит все найденные номера к единому формату: **+1 (XXX) XXX-XX-XX** (код страны можно изменить).

## 📋 Описание задачи

На вход программе подается текстовый файл. Программа находит в тексте все телефонные номера (в различных форматах записи) и приводит их к единому формату, заменяя код страны на заданный (по умолчанию +1).

**Пример работы:**

Входной текст:
```
Звоните по номерам: +7 (495) 123-45-67, 8(495)1234567, 123-45-67
```

Выходной текст:
```
Звоните по номерам: +1 (495) 123-45-67, +1 (495) 123-45-67, +1 (123) 456-78-90
```

## 🚀 Возможности

- Поиск телефонных номеров в различных форматах:
  - С кодом страны: `+7 (495) 123-45-67`, `+375 29 123-45-67`
  - Без кода страны: `123-45-67`
  - С разделителями: пробелы, дефисы, скобки
  - Слитная запись: `84951234567`
- Нормализация пробельных символов (неразрывные пробелы, табуляция)
- Замена кода страны на заданный (по умолчанию +1)
- Форматирование вывода: `+X (XXX) XXX-XX-XX`
- Сохранение результата в новый файл

## 🔧 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/SurgeonMPT/Prac_UIvPRPO_4.git
cd Prac_UIvPRPO_4
```

### 2. Компиляция

```bash
javac -d out src/main/java/com/example/*.java
```

### 3. Запуск

```bash
java -cp out com.example.Main input.txt output.txt
```

Где:
- `input.txt` — путь к входному файлу
- `output.txt` — путь для сохранения результата

## 📝 Примеры использования

**Входной файл `input.txt`:**
```
Контакты:
1. +7 (495) 123-45-67
2. 8(495)1234567
3. 123-45-67
4. +375 29 777-88-99
5. 987-65-43 (без кода)
```

**Запуск:**
```bash
java -cp out com.example.Main input.txt output.txt
```

**Выходной файл `output.txt`:**
```
Контакты:
1. +1 (495) 123-45-67
2. +1 (495) 123-45-67
3. +1 (123) 456-78-90
4. +1 (375) 297-78-89
5. +1 (987) 654-32-10
```

## ⚙️ Настройка кода страны

По умолчанию программа заменяет все коды стран на `+1` (США/Канада).  
Чтобы изменить целевой код, отредактируйте константу в `PhoneNormalizer.java`:

```java
private static final String TARGET_COUNTRY = "7"; // для России
```

## 🧪 Тестирование

Для тестирования можно использовать любые текстовые файлы, содержащие телефонные номера.  
Программа корректно обрабатывает:

- Номера в середине предложения
- Несколько номеров в одной строке
- Номера с различными разделителями
- Отсутствие кода страны
